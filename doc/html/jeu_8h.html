Permet la récupération des informations envoyés par le biais du joueur précédent. Ce en lisant dans le pipe du joueur précédent le num_fils. Quand l'information sera revenue, elle aura fait le tour.

int **  pipes,
struct_pendantjeu *  pendantjeulu

Fait passer les informations au suivant.

Permet la transmission des informations envoyés par le biais du joueur précédent. Ce en lisant dans le pipe du joueur précédent le num_fils. On l'envoie au suivant immédiatement. Cela nous permet d'appliquer la regle pour que le joueur revienne au point de départ si un autre joueur est arrivé sur la même case. Cette fonction est appellée lorsque ce n'est pas à num_fils de jouer.

Paramètres
num_filsInt - Correspond au numéro du fils souhaitant récupérer les informations.
pipesTableau des pipes.
pendantjeuluPointeur sur la structure contenant les informations du jeu qui vient d'être transmis par le voisin précédent.
Renvoie
Void - Termine en cas d'erreur

int **  pipes,
struct_retourjeu *  retourjeu,

)

Transmet les informations au père.

Quand les informations transmises ont fait le tour, on génère une structure contenant les informations dont le père aura besoin pour modérer. On lui envoie alors dans le pipe des fils vers le père.

Paramètres
num_filsInt - Correspond au numéro du fils souhaitant récupérer les informations.
pipesTableau des pipes.
retourjeuPointeur sur la structure contenant les informations du jeu qui vient d'être fait et ayant fait le tour.
resultatdeValeur du dé lancé. Utile pour le père afin de savoir si le fils doit rejouer ou non. En cas de double 6.
positionjoueurNouvelle position du joueur. Utile pour savoir si un joueur est arrivé au bout de sa course.
Renvoie
Void - Termine en cas d'erreur

int la_partie_est_interrompue ( struct_debuttour *  debuttourlu)

Verifie si la partie n'a pas été interrompue.

Dans le cas ou un joueur a gagné, nous considérons que la partie est finie. Dans le cas ou l'utilisateur choisi d'interrompre le programme cette fonction sera très utile pour permettre aux joueurs de se tuer même si ils n'ont pas fini de jouer.

Paramètres
debuttourluPointeur sur la structure lue au debut du tour.
Renvoie
Int - Renvoie 1 si la partie est termine ou interrompue, 0 sinon.
